# calc

Commandline expression calculator for Windows & Linux.

![image](https://github.com/radj307/calc/assets/1927798/725903a8-101f-46fa-8ea2-c5f0e6c54236)

## Features

- Multiprecision (Big numbers support)
- Variables
- Functions
- Bitwise Operators: [ `~` `|` `&` `^` `<<` `>>` ]
- Logical Operators: [ `==` `!=` `<` `>` `<=` `>=` ]
