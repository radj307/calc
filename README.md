# calc

Commandline expression calculator for Windows & Linux.

![image](https://github.com/radj307/calc/assets/1927798/72166e94-9aa9-4fb3-840f-b821b7b64a55)

## Features

- Multiprecision (Big numbers support)
- Variables
- Functions
- Bitwise Operators: [ `~` `|` `&` `^` `<<` `>>` ]
- Logical Operators: [ `==` `!=` `<` `>` `<=` `>=` ]
